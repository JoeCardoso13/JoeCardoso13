# Joe Cardoso

I'm a full-stack software engineer working at the edge of open source, developer tools, observability, and AI agents.

Most recently, I've been building with [NimbleBrain](https://nimblebrain.ai/) as a founding contributor, where my work focuses on agentic workflows for MCP server development, contributor onboarding, and the [mpak](https://mpak.dev/) registry ecosystem. I also contribute to Mozilla Firefox and previously co-created [Vispyr](https://vispyr.com), a self-hosted observability platform that combines continuous profiling with traditional telemetry.

## Selected Work

### NimbleBrain

Founding contributor working on agentic developer tooling for the NimbleBrain ecosystem.

- Built `nimblebrain-contributor`, a skill that onboards contributors through project discovery, issue selection, MCP server implementation, and registry announcement
- Architected `build-mcpb`, a multi-phase agentic workflow that guides an AI agent through the full lifecycle of building an MCP server integration
- Designed progressive-disclosure workflows that keep agent context anchored across sessions and implementation phases
- Contributed fixes across the `mpak` registry, including publish/announce validation and bundled native extension compatibility

### Mozilla Firefox

Open-source contributor to Firefox, with work spanning accessibility, DevTools, CSS architecture, and browser behavior.

- Fixed a long-standing accessibility bug where native HTML table cells were exposed with incorrect selectable state
- Completed a large DevTools Debugger CSS refactor across dozens of files, replacing generic image classes with namespaced debugger icon classes

### Vispyr

Co-created [Vispyr](https://vispyr.com), an open-source observability tool for continuous profiling and production performance analysis.

- Built CLI-driven AWS infrastructure deployment with CDK, CloudFormation, EC2, VPC peering, and IAM configuration
- Instrumented applications with OpenTelemetry, Prometheus Node Exporter, OTLP, gRPC, OpenMetrics, and HTTP telemetry paths
- Combined metrics, distributed traces, and profiles using Grafana Alloy, Prometheus, Tempo, and Pyroscope
- Extended Grafana with dashboards and flamegraph-centered workflows for code-level performance investigation
